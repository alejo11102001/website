<header>

La etiqueta <header> representa la cabecera de una sección o página. Suele contener el título, logotipo o el menú de navegación.


                <header>
                <h1>Bienvenido a MotoPlus</h1>
                <nav>...</nav>
                </header>

<nav>

<nav> define un conjunto de enlaces de navegación, como menús o barras de navegación internas.


                <nav>
                <ul>
                <li><a href="index.html">Inicio</a></li>
                ...
                </ul>
                </nav>

<section>

<section> representa una sección temática dentro de la página. Se usa para agrupar contenido relacionado.


                <section>
                <h2>Modelos destacados</h2>
                <p>Nuestras motos más vendidas.</p>
                </section>

<article>

<article> se usa para contenido independiente como entradas de blog, noticias, o publicaciones.


                <article>
                <h2>Nuevo modelo Vento RZ400</h2>
                <p>Con motor de 400cc y diseño deportivo...</p>
                </article>

<footer>

<footer> representa el pie de página de una sección o documento. Puede incluir autor, enlaces, o derechos.


                <footer>
                <p>© 2025 MotoPlus</p>
                </footer>
